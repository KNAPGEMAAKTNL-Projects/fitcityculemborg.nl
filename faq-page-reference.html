Lidmaatschap

Wat kost een lidmaatschap?
Fitness vanaf €24,50/maand (jaarabonnement). Ladies Only vanaf €20,50/maand. Kickboxing vanaf €19,95/maand. Dagpas: €7,00.
Wat is het inschrijfgeld?
Bij alle abonnementen betaal je eenmalig €17,00 inschrijfgeld.
Kan ik op elk moment opzeggen?
Bij Flex abonnementen opzeggen met 1 kalendermaand opzegtermijn. Na afloop vaste contracten geldt ook 1 maand opzegtermijn.
Kan ik mijn lidmaatschap pauzeren?
Neem contact met ons op om de mogelijkheden te bespreken.

Faciliteiten

Welke apparatuur hebben jullie?
Professionele apparatuur van Nautilus, SportsArt en Technogym.
Wat is de Ladies Only zone?
Een aparte trainingsvloer alleen voor vrouwen. Volledig uitgerust, volledige privacy.
Is er altijd personeel aanwezig?
Ja, tijdens alle openingstijden is er personeel aanwezig dat je gratis kan helpen.

Lessen

Wanneer zijn de kickboxinglessen?
Dinsdag 19:00-20:00, Donderdag 19:00-20:00, Zondag 10:00-11:00.
Kan ik een proefles doen?
Zeker! Je bent altijd welkom voor een gratis proefles.
Heb ik ervaring nodig voor kickboxing?
Nee, onze lessen zijn geschikt voor alle niveaus.
