## Analyse

### 1. Wat is de "lokale baseline"? (Hoe zien lokale Culemborg/Gelderland sportscholen er gemiddeld uit?)

De lokale sportschoolwebsites in de regio Culemborg/Tiel delen een aantal duidelijke patronen:

- **Ontwerp**: Functioneel maar gedateerd. WordPress-sites met standaardthema's (Astra, Starter Templates). Geen custom design, geen animaties, basale typografie. De sites "werken" maar inspireren niet.
- **Prijstransparantie**: Wisselend. Circle Fit en IZI Fitness tonen prijzen openlijk, maar Healthcenter, Training Factory en FamilyFit verbergen ze. Dit is een gemiste kans -- prijstransparantie verlaagt de drempel.
- **Trustsignalen**: Vrijwel afwezig. Geen enkele lokale concurrent toont Google-reviews, ledenaantallen, of certificeringen prominent. Social media links zijn het maximale. Dit is het grootste gat in de markt.
- **Hero's**: Statische foto's met tekst-overlay of eenvoudige sliders (Revolution Slider). Geen video, geen dynamische content. De hero's communiceren "wij bestaan" in plaats van "dit is waarom je hier wilt zijn".
- **CTA-strategie**: "Gratis proefles" of "Gratis dagpas" is de universele instap. Formulieren zijn basic (naam/email/telefoon). Geen directe online inschrijving, geen boekingssystemen behalve FamilyFit.
- **Positionering**: Elke sportschool kiest een niche: FamilyFit = families, Circle Fit = 30 minuten, Training Factory = vechtsporten, IZI = goedkoop, Healthcenter = wellness/revalidatie. Niemand probeert de "complete" sportschool te zijn.
- **Gemiddeld niveau**: Voldoende (een 5,5 op 10). De websites belemmeren niet actief, maar overtuigen ook niet.

### 2. Wat doen de nationale koplopers anders? (Wat scheidt best-in-class van gemiddeld?)

De nationale ketens opereren op een fundamenteel ander niveau:

- **Merkidentiteit**: Elke koploper heeft een helder, onderscheidend merkverhaal. TrainMore = urban lifestyle met gamificatie. Basic-Fit = "Feel Good Club" voor iedereen. SportCity = "Voor die gymlach". David Lloyd = premium wellness. Lokale clubs hebben een naam en een dienst, maar geen verhaal.
- **Video in hero**: SportCity gebruikt full-screen video (desktop en mobiel). TrainMore en Basic-Fit gebruiken professionele lifestyle-fotografie. Dit creëert een emotionele verbinding die statische foto's missen.
- **Trustsignalen gestapeld**: SportCity toont een 8,4 beoordeling van 228.874 leden -- dit is verpletterende social proof. Basic-Fit noemt 1.600+ clubs en gebruikt bekende ambassadeurs. TrainMore toont partnerschappen met Nike, Technogym, Life Fitness. Lokale clubs tonen niets.
- **Prijstransparantie als wapen**: Basic-Fit, SportCity en Circle Fit zetten prijzen in de hero of direct daaronder. Dit elimineert onzekerheid en versnelt conversie. De beste sites gebruiken vergelijkingstabellen met duidelijke tiers.
- **Conversie-architectuur**: Nationale sites hebben meerdere CTA's op elke pagina, gestroomlijnde checkout-flows, en een duidelijke funnel: gratis proefles > rondleiding > lidmaatschap. Lokale sites hebben een contactformulier en hopen op het beste.
- **Design als differentiator**: TrainMore's donkere thema met hot pink voelt premium. David Lloyd's paars-goud straalt luxe uit. Basic-Fit's oranje is energiek en herkenbaar. Ontwerp is niet decoratie -- het is positionering.
- **Promoties met urgentie**: "2 weken gratis", "Train gratis tot de zomer", "Luxe rugzak cadeau" -- nationale ketens begrijpen dat een tijdsgebonden aanbod conversie drijft.

### 3. Welke fouten maken de slechtste concurrenten?

De meest voorkomende fouten, gerangschikt op impact:

1. **Geen social proof**: De grootste en meest universele fout. Geen reviews, geen sterren, geen ledenaantallen, geen testimonials. In 2026 verwacht elke bezoeker bewijs dat anderen je vertrouwen. Zonder dit signaleer je "wij hebben niets om te laten zien" of erger, "wij weten niet hoe marketing werkt".

2. **Prijzen verstoppen**: Healthcenter, Training Factory en FamilyFit dwingen bezoekers om te zoeken, bellen of langskomen voor prijsinformatie. Dit creëert frictie en wantrouwen. Bezoekers die de prijs niet vinden, gaan naar een concurrent die die wel toont.

3. **Generieke hero zonder emotie**: Een stockfoto-achtige gymfoto met "Welkom bij [naam]" erboven zegt niets. Vergelijk dit met SportCity's "Voor die gymlach" met video of TrainMore's "Train gratis tot de zomer" met lifestyle-beeld. De beste hero's verkopen een gevoel, niet een faciliteit.

4. **Geen duidelijke positionering**: Sommige lokale sites proberen alles te zijn (fitness + groepslessen + sauna + fysiotherapie) zonder helder te maken waarom je specifiek bij hen moet zijn. Dit resulteert in een verwaterde boodschap.

5. **Verouderde technologie**: Revolution Slider, stock WordPress themes, geen mobiele optimalisatie, trage laadtijden. Dit signaleert verwaarlozing -- als de website verouderd is, hoe zit het dan met de apparatuur?

6. **Geen online boekingsflow**: De meeste lokale sites stoppen bij "vul dit formulier in en wij bellen terug". In 2026 verwachten consumenten directe online inschrijving of op zijn minst een realtime boekingssysteem.

7. **Geen urgentie of aanbod**: Lokale sites presenteren hun diensten als een statische menukaart. Er is geen reden om vandaag te handelen. Nationale ketens creëeren urgentie met tijdgebonden aanbiedingen en gratis proefperiodes.

## Kansen voor [FitCity]

Op basis van dit onderzoek liggen de grootste kansen voor [FitCity] in het gat tussen de lokale baseline en het nationale niveau:

- **Trustsignalen**: Als eerste lokale sportschool in Culemborg met prominente Google-reviews, een ledenaantal en eventueel een keurmerk springt [FitCity] er direct uit.
- **Prijstransparantie**: Prijzen direct op de homepage plaatsen. Dit is een concurrentievoordeel zolang lokale concurrenten dit niet doen.
- **Video in de hero**: Een korte video van de echte faciliteit en echte leden creëert direct meer vertrouwen en emotie dan elke stockfoto.
- **Helder merkverhaal**: Niet "nog een sportschool" maar een duidelijke positionering (bijv. gezondheidscentrum, community, bereikbaarheid) die in elke pagina doorklinkt.
- **Moderne online ervaring**: Directe online inschrijving, realtime rooster, app-integratie -- dit is wat nationale ketens bieden en lokale concurrenten missen.

## CSS Layout Patterns That Make the Best Ones Look Professional

### Pattern 1: Full-Bleed Hero with Gradient Overlay
```css
.hero {
  position: relative;
  min-height: 720px;
  display: flex;
  align-items: center;
  overflow: hidden;
}
.hero video {
  position: absolute;
  inset: 0;
  width: 100%;
  height: 100%;
  object-fit: cover;
}
.hero::after {
  content: '';
  position: absolute;
  inset: 0;
  background: linear-gradient(180deg, transparent 0%, #141414 100%);
  pointer-events: none;
}
.hero-content {
  position: relative;
  z-index: 1;
  max-width: 600px; /* left-aligned, not centered */
}
```

### Pattern 2: Contained Content in Full-Bleed Sections
```css
.section { /* full-bleed background */
  padding: 4rem 1rem; /* mobile */
}
@media (min-width: 1024px) {
  .section {
    padding: 8rem 2.5rem; /* desktop: generous spacing */
  }
}
.section-inner {
  max-width: 1350px; /* TrainMore reference */
  margin: 0 auto;
}
```

### Pattern 3: Asymmetric Two-Column (Not 50/50)
```css
.split-section {
  display: grid;
  grid-template-columns: 1fr;
  gap: 2rem;
}
@media (min-width: 768px) {
  .split-section {
    grid-template-columns: 5fr 7fr; /* asymmetric: ~42% / ~58% */
  }
  .split-section.reversed {
    grid-template-columns: 7fr 5fr;
  }
}
```

### Pattern 4: CTA Button with Hover Inversion
```css
.cta-primary {
  background: #FFE303;
  color: #141414;
  font-weight: 700;
  text-transform: uppercase;

```

### Pattern 5: Equipment Logo Marquee
```css
.logo-strip {
  display: flex;
  gap: 3rem;
  overflow-x: auto;
  scroll-snap-type: x mandatory;
  -webkit-overflow-scrolling: touch;
  mask-image: linear-gradient(90deg, transparent, #000 10%, #000 90%, transparent);
}
.logo-strip img {
  height: 2rem;
  object-fit: contain;
  filter: brightness(0) invert(1); /* white logos on dark bg */
  opacity: 0.6;
  transition: opacity 250ms;
  scroll-snap-align: center;
}
.logo-strip img:hover { opacity: 1; }
```
